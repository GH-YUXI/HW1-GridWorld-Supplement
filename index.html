隨機策略逐步紀錄

    操作說明

    1. 選擇 n（5 到 9）並建立網格。
    2. 設定步驟獎勵 Reward 與折扣因子 γ。為確保收斂，γ 必須滿足 0 ≤ γ < 1。
    3. 切換模式後，點擊格子設定起點、終點、障礙物。
    4. 障礙物數量最多為 n-2，可少於 n-2。
    5. 按下「執行 Value Iteration」後，格子會顯示最佳狀態價值 V(s)、最佳策略箭頭與一條抽樣最佳路徑。
    6. 按下「執行隨機策略」後，系統會以四個方向等機率的隨機策略進行 Policy Evaluation，並顯示一條隨機策略抽樣軌跡。
    7. 按下「逐步播放隨機策略」後，系統會逐格顯示這次隨機抽樣如何移動，包含目前位置、隨機選到的方向與近期步驟紀錄。
    8. 若起點無法到達終點，系統會先提示錯誤，不會執行策略計算。

    請先建立網格並完成設定。
